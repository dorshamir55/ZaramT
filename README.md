# ZaramT - Android app

Developed using Java in an Android studio.

Do you have a dilemma? Come share and flow!
The app allows sharing daily dilemmas.
And when the dilemma's time is up - you flow on the most chosen answer!

Included: Fragments, MVVM Architecture, ViewModel, Repository, LiveData, FireBase, Room & Cache, Recyclerview, Shared Preferences, Asynchronous Programming, Services, Broadcast Receivers.
